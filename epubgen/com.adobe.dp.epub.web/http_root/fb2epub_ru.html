Конвертация FB2 в EPUB

Файл в формате FB2:

Выберите файл в формате FB2. Это может быть как текстовый (.fb2), так и сжатый (.fb2.zip) файл. Совместимые версии формата: 2.0 и 2.1.

Все структурные элементы FB2 распознаются конвертером, включая таблицы и стили. С другой стороны, конвертация атрибутов (таких как имя программы, создавшей FB2) осуществляется только частично.

Здесь могут быть заданы дополнительные стили - либо как CSS либо как FB2 (в этом случае используются только стили из документа-шаблона, содержание этого документа несущественно). Примеры шаблонов:

  • Пример нестандартного шаблона
Транслитерировать заглавие, автора и другие атрибуты книги

Выберите эту опцию, если ваше устройство для чтения не русифицировано

Сам текст книги не нуждается в транслитерации, т.к. конвертер включает в документ шрифты, необходимые для отображения текста программой для чтения. Однако атрибуты книги (заглавие, автор, содержание) отображаются с помощью шрифтов установленных на самом устройстве и, если эти шрифты не содержат кириллицы, атрибуты будут отображаться "квадратиками".

Использовать ссылку на файл FB2 (URL)
